# Launch MVP

## V1 (Ship Now)
- Auth (`register`, `login`, `me`)
- Project create/list/update/delete
- Acapella upload + analysis
- Style-select generation
- Preview + download (`beat-audio`, `mix-audio`)
- Responsive web flow (mobile browser usable)

## V1 Guardrails
- No major UI branching
- Keep one generation path
- Keep one backend API process
- Keep one generation worker path (current synchronous flow is acceptable for v1)

## V1.1 (Immediately After Launch)
- Queue-backed generation worker
- Asset metadata UI + search
- Hybrid style blending presets
- Better observability and analytics funnel

## V2
- Subscription + credits
- Pack marketplace
- Mobile wrapper distribution (Capacitor)
